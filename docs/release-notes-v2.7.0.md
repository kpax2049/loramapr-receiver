# LoRaMapr Receiver v2.7.0 (Public Launch Hardening and Docs Polish)

Release date: 2026-03-12

## Highlights

- Polished public install documentation around the two supported paths:
  - Raspberry Pi appliance image (recommended)
  - Existing Debian-family Linux / Raspberry Pi OS package install
- Reworked README and docs index to make first-time setup flow easier to follow.
- Improved local portal wording for first-time operators:
  - clearer pairing and next-step language
  - clearer issue/attention framing
  - reduced implementation-heavy phrasing
- Consolidated support and troubleshooting workflow for both install paths.
- Updated reviewer smoke-test guidance for public launch validation across
  install, pairing, diagnostics export, and recovery flows.
- Added v2.7 launch-hardening scope/spec for ongoing public-facing consistency.

## Scope and Safety

- No major runtime feature additions.
- No install-path behavior changes.
- Existing lifecycle/update/support/automation behavior remains intact.
- Focus of this release is user-facing clarity, documentation coherence, and
  launch polish.
